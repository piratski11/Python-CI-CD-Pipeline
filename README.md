CI/CD Pipeline для Docker-приложения (GitLab CI)

Описание

Проект демонстрирует полный цикл CI/CD для Python-приложения с использованием GitLab CI/CD и Docker.

Pipeline автоматически:

- запускает тесты;
- собирает Docker-образ;
- публикует образ в Docker Hub;
- выполняет деплой на удаленный сервер по SSH.

---

 Используемые технологии

- GitLab CI/CD
- Docker
- Docker Hub
- Python 3.11
- Pytest
- SSH
- Linux

---

 Pipeline

 1. Тестирование

Перед сборкой контейнера автоматически запускаются тесты.

Используется:

- pytest

---

 2. Сборка

После успешного прохождения тестов:

- собирается Docker-образ;
- присваивается тег;
- создается latest.

---

 3. Публикация

Образ автоматически отправляется в Docker Hub.

---

 4. Деплой

По SSH выполняется:

- подключение к серверу;
- остановка старого контейнера;
- загрузка новой версии;
- запуск контейнера;
- очистка неиспользуемых образов.

---
